# Roadmap

> This roadmap reflects strategic priorities and may evolve based on security
> requirements, ecosystem changes, community feedback, and operational learnings.

---

## 2026 Q2 — Governance Hardening

| Priority | Item |
|---|---|
| **High** | Stabilize enterprise governance and release process |
| **High** | Add code coverage reporting and CI badge integration |
| **Medium** | Expand operational runbooks for incident handling and recovery |
| **Medium** | Implement automated changelog generation from Conventional Commits |

## 2026 Q3 — Production Readiness

| Priority | Item |
|---|---|
| **High** | Production deployment guides for managed infrastructure (Docker, cloud) |
| **High** | Observability documentation: structured logging, metrics, alerting |
| **Medium** | Dependency compatibility matrix (Node.js, npm, Solana SDK versions) |
| **Medium** | API versioning strategy and deprecation policy |

## 2026 Q4 — Contributor Ecosystem

| Priority | Item |
|---|---|
| **High** | Formalize release security review checklist |
| **Medium** | Contributor automation: auto-labeling, stale issue management, triage bots |
| **Medium** | Long-term support (LTS) and maintenance cadence commitments |
| **Low** | Contributor leaderboard and recognition program |

## Long-Term Vision

| Area | Direction |
|---|---|
| **Multi-network** | Mainnet-beta support, multi-chain abstraction layer |
| **Compliance** | Enterprise audit readiness mappings (SOC 2 alignment) |
| **BYOA** | Advanced guardrails, optional policy packs, rate limit tiers |
| **Ecosystem** | Plugin marketplace for community-contributed strategies |
| **Observability** | OpenTelemetry tracing, Prometheus metrics export |
